### ⬇️ Clone
```sh
git clone https://github.com/templates-ecosystem/template-vike-lite-solid-daisyui-hono.git
```

### ⚙️ Install
```sh
yarn
```

### 🚀 Start
```sh
yarn dev
```

### ▲ Deploy/Config Vercel
See [Vercel Guide](/vercel/README.VERCEL.md)

### ▲ Deploy/Config Docker
See [Docker Guide](/docker/DOCKER.md)

> **Note:** `standaloner`, with the option `bundle=true`, bundles the server as a single file `dist/server/index.mjs`.
So dependencies like `hono` and eventually `@hono/node-server` can be installed as dev dependencies instead of dependencies.
